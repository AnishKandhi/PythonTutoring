Sets

A set is another data structure that is a collection of values, and they are iterable in the same way as lists. However, they are unordered, meaning they can't be indexed, and they contain no duplicates. Adding values that are already in the set will be automatically filtered out by the set to remove duplicates. Sets consist of brackets ({}). 


set1 = {5, 3, 10, 4}

Set Functions

  • add(item): appends a value to the end of a set
  • remove(item): removes value from the set, throws error if the value is not found
  • a.union(b) or a | b: returns new set consisting of all values from both sets with duplicates filtered out.
  • a.intersection(b) or a & b: returns a new set with common values between sets a and b
  • a.difference(b) or a - b: returns a new set with the unique values of set a that set b doesn't have
  • a.intersection(b) or a & b: returns a new set with common values between sets a and b
  • a.symmetric_difference(b) or a ^ b: returns a new set with the distinct values from both sets
  • clear():removes all items from set

a = {5, 3, 10, 4}
b = {10, 2, -3.5, 3}
a.add(-1) #{5, 3, 10, 4, -1}
a.remove(-1) #{5, 3, 10, 4}
print(a.union(b)) #{5, 3, 10, 4, 2, -3.5}
print(a.intersection(b)) #{10, 3}
print(a.difference(b)) #{5, 4}
print(a.symmetric_difference(b)) #{5, 4, -3.5, 2}
print(a.clear()) #{}
