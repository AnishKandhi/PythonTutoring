Tuples

Tuples also hold a collection of values similar to a list. The main difference is that they use parentheses to store elements, and they are also immutable. So unlike lists, the elements of a tuple can not be manually changed, but they can still be indexed and sliced. They can also be added and multiplied. 


coords = (3, 2)
print(coords[0]) #3
print(coords  + (4, 6))
#(3, 2, 4, 6)
print(coords * 3)
#(3, 2, 3, 2, 3, 2)
coords[0] = 5 #ERROR
#Tuples are immutable

Traversing Through a List of Tuples

Iterating through a tuple is essetially the same as iterating through a list. However, when you have a list of tuples, there is a more convenient way to traverse. This is called "tuple unpacking". It only works where there is a list of two-element tuples, and it utilizes two loop variables, one for each element. 


coords = [(5,2), (10, -6), (3, 4), (1, 0)]
#--------Basic for loop--------
for pair in coords:
    print(f"({pair[0]}, {pair[1]})", end=" ")
#(5, 2) (10, -6) (3, 4) (1, 0)
---------Tuple Unpacking----------
for num1, num2 in coords:
    print(f"({num1}, {num2})", end =" ")
#(5, 2) (10, -6) (3, 4) (1, 0)
