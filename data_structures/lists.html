Lists

A list is another data type in python that can store multiple values. For example, a list can store a collection of integers, strings, booleans, other lists, or a combination of them. Lists consist of square brackets ([]) and values inside of them. 


int_list = [5, 3, 2, 10, -2]
str_list = ["str1", "str2", "str3"]
bool_list = [True, True, False, False]
list_2D = [[5,4], [3, -9], [108, 312]]

Lists Mutability

Lists are mutable, meaning their values can be manually changed by accessing each element by its index. Just like strings, index numbers for lists go from 0 . . . length - 1. 


int_list = [5, 4, 10]
int_list[1] = 0
#[5, 0, 10]
---------------------------
list_2D = [5, [10, 6], 14]
list_2D[1][1] = -1
#[5, [10, -1], 14]

List Functions

Lists have several important in-built functions. The most important include:

  • len(list_name): gets the number elements in the list
  • append(val): adds value to end of the list
  • remove(val): removes first occurrence value from the list
  • extend(list): appends elements of list into existing list
  • pop(index): removes and returns value at the index
  • reverse(): Reverses the elements of the array from their current order
  • sort(): Reorders the elements to ascending order for an integer list and alphabetical order for string list. 

int_list = [5, 3, 10, 13]
int_list2 = [2, 9]
print(len(int_list)) #4
int_list.append(6) #[5, 3, 10, 13, 6]
int_list.remove(3) #[5, 10, 13, 6]
int_list.extend(int_list2)
#int_list: [5, 3, 13, 6, 2, 9]
print(int_list.pop(3)) #6
#int_list: [5, 3, 13, 2, 9]
int_list.reverse() #[9, 2, 13, 3, 5]
int_list.sort() #[2, 3, 5, 9, 13]
int_list.sort(reverse = True) #Descending order
#int_list: [13, 9, 5, 3, 2]
        

--------List Slicing--------
#works the same way in lists as strings
nums = [5, 3, 8, 2, -9, 42.3]
print(nums[2:4]) #index 2-3
#Output: [8, 2]

print(nums[1 : 5]) #index 1-4
#Output: [3, 8, 2, -9]

Traversing through lists

Traversing through lists using loops is one of the most important concepts in programming. It works basically the same way as looping through strings. We can either directly loop through each element using a for each loop or access them by index using range(). 


num_list = [4, -2, 10.9, 9, -9.1]
------For each loop-------
for num in num_list:
    print(num, end = "  ")
#Output: 4  -2  10.9  9  -9.1

-------index loop-------
for index in range(len(num_list)): #0 to len-1
    print(num_list[index], end = "  ") 
#Output: 4  -2  10.9  9  -9.1

2D list traversals

2D lists are lists inside of another list. Traversing through them involves a nested for loop. The loops can either be for each loops or index-based loops just like the previous example. Here is a breakdown: 


int_list = [[4, -2, 10.9], [5.6, 0, -3], [5, 9]]

#------For each method--------
for sub_list in int_list: #grabs each inner list
    for num in sub_list:  #each val in the inner list
        print(num, end = " ")
#Output: 4  -2  10.9  5.6  0  -3  5  9
#--------Index method---------
for i in range(int_list): #index of each sub-list
    for j in range(len(int_list[i])): #each element
        print(int_list[i][j])
#Output: 4  -2  10.9  5.6  0  -3  5  9

Example Application

Let's say we want to write a function count the number of even values in a list that consists of both integers and sub-lists. We would need to traverse using a set of for loops, and use a conditional to check whether the current element is a list or an integer, so we know how to check if it's even. To do this, we would need to utilize the type() function, which gives the data type of a value. 


def count_evens(list):
    count = 0
    for val in list:
        if type(val) == int:
            if (val % 2 == 0):
                count+=1
        elif type(val) == list:
            for x in val:
                if x % 2 == 0:
                    count+=1
    return count
