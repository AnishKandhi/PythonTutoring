Dictionaries(Dict)

A dictionary in python is a collection of key-value pairs, seperated by colons(:), where ethe key is on the left and the value is on the right. The keys much be distinct from each other and immutable data types (ints, strings, tuples), while the values can be any data type. 


student = { 
"name": "Tom", 
"grade": 7, 
"id": 593
}

Dictionary Manipulation

Dictionary values can be accessed trough their keys in two ways. We can either use square brackets ([]) with the key name inside or call the get() function. Adding or modifying a dictionary element involves the use of square brackets. Removing a key-value pair can be done with the del keyword or pop() function. 


student = { 
"name": "Tom", 
"grade": 7, 
"id": 593
}

print(student["name"]) #Tom
print(student.get("id")) #593
student[grade] += 1 #"grade": 8
student["class"] = "History" #adds new pair
del student["id"] #removes pair with key-"id"
student.pop("name") #Removes pair with key-"name"

Other Dictionary Functions

Note: A view object is an iterable data structure similar to a list, but it updates in live time, and can not be indexed.

  • keys(): returns a view obejct of all the keys
  • values(): returns a view object of all the values
  • update(dict): adds key-value pairs of other dictionary into existing
  • items(): returns a view object of tuples containing key-value pairs
  • clear():removes all items form dictionary

student = { 
"name": "Tom", 
"grade": 7, 
"id": 593
}

print(student.keys()) 
#Output: dict_keys(["name", "grade", "id"])

print(student.values())
#Output: dict_values(["Tom", 7, 593])

print(student.items())
#Output: dict_items([("name", "Tom"), ("grade": 7), 
("id": 593)])

student.update({"Car": "BMW", "Major: Math"})
#student: {
"name": "Tom", 
"grade": 7, 
"id": 593,
"Car": "BMW",
"Major": "Math"
}

Traversing Through a Dictionary

Dictionary traversals can be done with a simple for loop. Traversing through full key-value pairs involves the use of the items() function from earlier, since we need to traverse over a collection of tuples. 


student = { 
"name": "Tom", 
"grade": 7, 
"id": 593
}

#Printing keys
for key in student:
    print(key, end = "  ") 
#name  grade  id
-----------------------------------
#Printing values
for key in student:
    print(student[key], end="")
#Tom  7  593
-----------------------------------
#Key-value Pairs Iteration
for key, value in student.items():
    print(f"{key}:{value}")
