Functions

A function is a segment of code that can be reused with a method call. We have already used several in-built functions, including print(), len(), range(), and int(). So calling a function requires parentheses after the name of the function, and parameters to pass into the parentheses if needed. We can also define our own function and call it just like how we call the in-built ones, and python makes it very simple.

Defining Functions

Functions are defined using the 'def' keyword, followed by the name of the function, parentheses, and placeholder parameters in the parentheses. Inside the function is where the code for the function is written. 


#Without parameters

def print_int():  #function declaration
    print(7)
print_int() #function call
#7 is Output
------------------------------------------
#With parameters

def print_x(x): #x serves as parameter
    print(x)
print_x(10) #10 Output

#Notes:
#10 substitutes in for x
#So what we did with x is the same for 10

Return Values

Functions do not simply print things. They can return values of all data types, which can then be used later to be stored in other variables or Output. The return key word specifies which value is returned as a result of the function call. 


def get_len(string):
    return len(string)
length = get_len("python") #storing return val
if (get_len(length) % 2 == 0): 
    print("Is even")
else:
    print("Is odd")
#Output: Is even

#Notes:
#get_len() returns 6
#Since 6 is even, first condition is true

Default Parameters

The function parameters we have used have been non-default thus far, meaning we have to call the function with the paramter included in the parentheses or we will get an error. We can also declare default paramters, where calling the function without the parameter included will simply auto-fill the paramter value with we default it to. Here is how: 


def add(a, b):
    return a + b
print(add(4)) #ERROR, 2 parameters needed
-----------------------------------------
def add(a, b=5):
    return a + b
print(add(4)) #b defaults to 5

#Output: 9
-----------------------------------------
def add(a=5, b):
    return a + b
print(add(4)) #ERROR
#Default parameter should come after non-default
----------------------------------------
def add(a=5, b=4):
    return a + b
print(add()) #a and b default to 5 and 4
#Output: 9
