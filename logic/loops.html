Loops

A loop is a segment of code that keeps repeating over and over again until a specific condition is met. There are two types of loops: for loops and while loops. A for loop uses a local, temporary variable to guide how many times (iterations) he loop runs. A while loop can hold any kind of condition to guide how the loop executes. Both will be explored in depth.

For Loops

For loops use a temporary variable built into the loop. This variable can be used in the loop itself and to indicate how many repetitions/iterations the loop runs for. This variable is oftne combined with the range() function, which specifies the bounds of the variable under which the loop executes. The loop var starts at the left bound and increments by 1 until it reaches the right bound. See code snippet for breakdown. 


for i in range(1, 4): #i goes from 1 to 3
    print(i, end = " ")

#Output: 1 2 3      
#i goes from 1 to 4, exclusive
#Once it reaches 4, the loop stops 
-------------------------------------
for i in range (5)
    print(i, end = " ")

#Output: 0 1 2 3 4
#left bound is 0 by default
#If only one value passed into range(),
#loop var goes from 0 to that num, exclusive

Backwards Loops

The third parameter/value of the range() function is steps, or how much the loop var should increment each iteration. By default, the step is 1. By setting the step to a negative number, like -1, the variable decrements backwards. 


Upward increments:

for i in range (5, 21, 5):
    print(i, end = " ")

#Output: 5 10 15 20
#i starts at 5 and jumps by 5 each round
        

Backwards loop:
for i in range (4, 1, -1):
    print(i, end = " ")

#Output: 4 3 2 
#i starts at 4 and decrements by 1
#loop stops when i reaches 1

Loop Traversals

Loops are effective at traversing through elements of strings, lists, and other data structures. They can search strings character by character, either by directly accessing characters or by accessing them through their index. 


#Accessing directly
lang = "Python"
for char in lang:
    print(char, end = " ")

#Output: P y t h o n
-------------------------------------
#Accessing through index
for i in range(len(lang)): #0 to last index
    print(lang[i], end = " ")

#Output: P y t h o n
#Remember the last index of a string is length - 1,
#So we loop from 0 to the length exclusive

Nested for loops

Nested for loops are loops inside another outer loop. In this case, each iteration of the outer loop consists of a full course of the inner loop. The outer loop moves onto the next iteration once the inner loop fully breaks. In the example below, the outer loop waits for the inner loop to finish (when j = 2) before i increments. The entire block ends once the outer loop ends (when i = 3). 


for i in range(3): 
    for j in range(2):
        print(i + ":" + j, end = " ")

#0:0 0:1 1:0 1:1 2:0 2:1

While loops

While loops are the other type of loops. Like a for loop, a while loop runs as long as the condition as not been met yet. However, while loops don't use a temporary loop variable like for loops. If you want to use one, you have to manually declare the variable. While loops are useful in specifying any kind of condition, not just ones using a loop var. 


#Using a while loop like a for loop
x = 0
while x < 3:
    print(x, end = " ")
    x += 1  #incrementing x

#Equivalent form using for loop
for x in range(3):
    print(x, end = " ")

#Output for both: 0 1 2
        

#Using a while loop like a for loop
x = 0
while x < 3:
    print(x, end = " ")
    x += 1  #incrementing x

#Equivalent form using for loop
for x in range(3):
    print(x, end = " ")

#Output for both: 0 1 2

While loops for other purposes

While loops are more diverse than for loops in that they can specify any kind of condition. For example, the code snippet below shows a mini guessing game, where the code repeatedly prompts the user to enter a number, and the program stops when the user enters the correct number. 


#Guessing game
guess_num = 47  #num to be guessed
num = int(input("Enter a number: "))
while num != guess_num: #condition
    num = int(input("Enter a number: "))
print("You win!")
#If loop ended, the user must have guessed right
