Control Flow

Control flow in programming is setting conditions under which each line of code should run. Previously, we have seen that code runs line-by-line in chronological order. When we implement conditional statements, the logic also runs top to bottom, so statements should be carefully placed in strategic order.

Boolean Logic

We learned that boolean values can hold True or False. This attribute is useful in control flow, where conditions rely on boolean values when deciding whether to excute a statement. There are several comparison operators and logical operators, including ==, !=, "and", and "or".

  • "==" : Compares two values for equality, different from assignment operator "=";
  • "!=" : Compares two values for inequality; returns true if not equal
  • "and" : Checks if both boolean statements are true, and returns true if so;
  • "or" : Checks if one boolean statement is true, and returns true if so;;

x = 5
y = 7
z = 5.0
is_cool = True
is_windy = False
is_dusty = True
print(x == y) #False
print(x != y) #True
print(x == z) #True
print(is_cool and is_windy) #False
print(is_windy or is_cool) #True
print(is_cool and is_dusty) #True

If-Statements

If-statements, or conditional statements, are specific lines of code that set a condition under which code should or should not execute. The inside of an if-statement contains a boolean expression. If that expression returns True, then only will the code inside the if-block run. Otherwise, code in the else-block runs if it exists. If/else statement have a colon next to the condition, and code inside of it must be indented to the right.


is_py = True      #some bool var
is_js = False
if is_py:  #since is_py = True, the conditon is met
    print(True)
else:
    print(False)
#the block prints True
if is_py and is_js:
    print(True)
else:
    print(False)
#False Output since is_js is False
#and means both have to be True
-----------------------------
if not is_js: 
    print(True)
else:
    print(False)
#The 'Not' reverses the boolean value
#So the block prints True
        

x = 10 
y = 17
if x != y:
    print(True)
#Statement prints True since x != y is true
if y - 10 == x:
    print(True)
else:
    print(False)
#False is Output

Structuring Logic

When you have mutliple subsequent if-statements working together, structuring your conditions logically is very important. The 'elif' keyword also becomes useful, which makes its condition tested if the if-statement and elif statements above it were not executed yet. 


-----WRONG WAY-----
num_grade = 100
letter_grade = ""
if grade >= 90:
    letter_grade = "A"
if grade >= 80:   #runs regardless
    letter_grade = "B" #runs regardless
if grade >= 70:  #runs regardless
    letter_grade = "C"
else:  
    letter_grade = "F"
print(letter_grade)
#Will ALWAYS print C
#Letter_grade keeps reassigning each if statement
        

-----CORRECTED WAY-----
num_grade = 100
letter_grade = ""
if grade < 70:
    letter_grade = "F"
if grade >= 70:   #runs regardless
    letter_grade = "C" #runs regardless
if grade >= 80: #runs regardless
    letter_grade = "B"
if grade >= 90:
    letter_grade = "A"
print(letter_grade)
#Grade starts low but keeps reassigning to higher grade
        

-----RIGHT WAY-----
num_grade = 100
letter_grade = ""
if grade >= 90: 
    letter_grade = "A"
elif grade >= 80:  #only runs if above doesn't
    letter_grade = "B"
elif grade >= 70:  #only runs if above doesn't
    letter_grade = "C"
else:  #only runs if all other statements don't
    letter_grade = "F"
print(letter_grade)

Nested If-Statements

You can also have if-statements within other if-statements. In this case, if the out if statement executes, then only does the inner if-block execute. Otherwise, the code in the outer-else block executes, if it exists. 


is_easy = True
num = 6
if is_easy: #Inner statements run only if true
    if (num % 2 == 1)  #false since 6 is even
        print("1") 
    else:
        print("2") #runs if prev is false
else:
    print("3") #runs if outer statement is false
#2 is Output

Match Statements

Match statements are a shortened, more concise form of if-statements. However, they only work for checking if a variable is equal to a certain value. For example, it can check if an integer var equals a specific digit or if a string var equals a certain character. A match statement consists of case statements for possible values, and the code inside the case executes if the condition is met, or the case matches the value. The case _ only runs if none of the cases match. 


char = 'd'
match char:
   case 'a':  #checks if char == 'a'
       print('1')
    case 'b': 
       print('2')
    case 'c':
       print('3')
    case 'd':  #matches
       print('4') #Output: 4
    case_: #in case none match
       print("none")

Other String Functions

  • len(Strings): Gets the number of characters in a string
  • substring(start_index, end_index): Slices the string from start_index (inclusive) to end_index (exclusive). The end index is the length of the string by default.
  • find(str): gets the index of the string or char in the parameter; returns -1 if not found.
  • replace(str1, str2): Replaces the substring str1 with str2 in its old position.
  • upper(str), lower(str): Returns an uppercase or lowercase version of the string, respectively.
  • startswith(str), endswith(str): Returns true/false based on whether string starts with or ends with the substring, respectively. 

lang = "Python"
print(len()) #6
print(lang.find('t')) #2
print(lang.find('z')) #-1
print(lang.startswith("Py")) #True
print(lang.endswith("On")) #False - Case Sensitivity
print(lang.upper()) #PYTHON
lang.replace("yt", "oa") #lang is now poathon
