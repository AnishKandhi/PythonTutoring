Classes & Objects: Inheritance

Inheritance is where child classes, or subclasses, inherit the attributes and methods of the parent class (base class), and can use them without ever actually defining the methods/attributes.

Basic Syntax

Child classes can inherit the properties of parent classes by passing the parent class name into the parameters of the child class. By using this, the child class becomes the subclass, and the parent becomes the base class. Methods and variables of the parent class can automatically be used by the child. 


class Parent:
    def __init__(self):
       self.number = 70
    def inherited_method(self):
        return "This is from the parent class"

class Child(Parent):  #Inheritance
    pass     
        

#-------Base Class Accessing------
child = Child()
print(child.number) #70
print(child.inherited_method())
#Output: This is from the parent class

Attribute Overriding

The tricky part comes in when the child class has attributes and methods with the exact same name and signature as the parent class. Which one's method or variable would be used? In all cases, the child class's new values override the parent's exisiting methods and attributes, so child classes would, by default, use the new values. If we want to still access the parent class's overriden values, we could use the super() function. 


#-----Attribute Overriding------
class Parent:
    def __init__(self): 
        self.num = 100

class Child(Parent):
    def __init__(self):
        super().__init__() #calls parent's constructor
        self.num = 75
        

-----Resolving Value Conflicts-----
parent = Parent()
child = Child()
print(parent.num) #100
print(child.num) #75

Method Overriding

Methods are overidden in the same way and parent methods could still be called using super(). 


#-----Method Overriding------
class Parent:
    def get_val(self):
       return 50 

class Child(Parent):
    def get_val(self):  #overidden
       return super().get_val() + 25
        

-----Resolving Method Conflicts-----
parent = Parent()
child = Child()
print(parent.get_val()) #50
print(child.get_val()) #75
