Variables in Python

A variable is a name used to store a specific value. Variables can be in the form of integers, strings, boolean, floats, and data structures like lists and dictionaries. We will learn about each of these data types further in this lesson. In the image to the right, x stores an integer, name stores a string, and is_tall stores a boolean value the variable x.


x = 7
name = "John"
is_tall = True

Integer (int)

An integer is a whole number. A variable of data type int can only store a whole number and not a decimal. Storing a decimal number in an integer results in a value error. Remember that the "=" sign is an assignment operator, meaning it assigns a value to the variable. 


x = 25
y = 100
number = 65
num = 6.5 #Not an int
print(y + x) #125
print(y - x) #75 
print(y * x) #2500
print(y / x) #4

Float

A float is a decimal number. A variable of data type float can store both a real whole and decimal number. Printing a whole number variable of type float prints its floating point value. A float can be casted to an int using the int() function, and vice versa. Example in the code snippet shown.


float_num = 5.6
int_num = int(float_num)
float_val = float(5)
print(float_num) #5.6
print(int_num) #5
print(float_val) #5.0

Boolean (bool)

A boolean is a true/false value. A variable of data type bool can only store the value True or False. Printing a boolean variable will either output True or False, depending on which it is assigned to.


is_weekend = True
is_holiday = False
print(is_weekend) #True

String (str)

A string is a value that contains quotes. A variable of data type str can store a mix of letters, numbers, and symbols in quotes. Example strings include words, names, and quoted digits.


name = "Rick"
character = 'B'
num = str(5) #'5'
print(name, character) #Rick B

String Manipulation

  • String Concatination: Strings can be added together to create a new string using the + operator
  • Indexing: Individuals characters of a string can be accessed by using string_name[*index num*] where the first character has index 0, and the index increments for each subsequent char. The last char has an index of -1, and the index of each previous char counts down.
  • Slicing: Subsets of a string can be accessed using string_name[start_index : end_index + 1: increment_amount]. We use end_index + 1 because the slicing parameter only goes from start_index to end_index exclusive.
  • f strings: Values of variables do not have to be concatenated, but could be directly placed into an existing string using brackets ({}) with the key character 'f'. 

lang = "Python"
use = "coding"
print(lang + " " + use) #Python coding
print(f"Coding in {lang}") #Coding in python
print(lang[3]) #h
print(lang[-1]) #n
print(lang[1:4]) #yth
print(lang[2:5:2]) #to

Other String Functions

  • len(Strings): Gets the number of characters in a string
  • substring(start_index, end_index): Slices the string from start_index (inclusive) to end_index (exclusive). The end index is the length of the string by default.
  • find(str): gets the index of the string or char in the parameter; returns -1 if not found.
  • replace(str1, str2): Replaces the substring str1 with str2 in its old position.
  • upper(str), lower(str): Returns an uppercase or lowercase version of the string, respectively.
  • startswith(str), endswith(str): Returns true/false based on whether string starts with or ends with the substring, respectively. 

lang = "Python"
print(len(lang)) #6
print(lang.find('t')) #2
print(lang.find('z')) #-1
print(lang.startswith("Py")) #True
print(lang.endswith("On")) #False - Case Sensitivity
print(lang.upper()) #PYTHON
lang.replace("yt", "oa") #lang is now poathon
